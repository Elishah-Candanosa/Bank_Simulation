# Simulación de un Banco

* **Autor:** Elishah Candanosa

Este repositorio contiene el proyecto "Simulador de un Banco", desarrollado en Python utilizando Jupyter Notebooks. A través de este simulador, es posible modelar y analizar el comportamiento de la atención al cliente en un entorno bancario, permitiendo realizar experimentaciones sobre el flujo de personas y el tiempo de atención.

## Descripción del proyecto

El simulador permite realizar las siguientes acciones:
- **Iniciar una nueva simulación:** Comienza un nuevo ciclo de simulación del banco.
- **Consultar resultados pasados:** Muestra los resultados obtenidos en la última simulación realizada.
- **Exportar resultados:** Genera y exporta un archivo txt con los resultados de la última simulación.
- **Salir del programa:** Permite finalizar la ejecución del simulador.

Además, incorpora controles interactivos de usuario, como sliders y botones, para parametrizar la simulación. Por ejemplo:
- Configurar cuántos minutos y segundos adicionales el banco permanecerá abierto.
- Analizar y ajustar los parámetros con intuición visual.

## Requisitos previos

- **Python 3**
- **Jupyter Notebook** (parte de Jupyter Lab o Anaconda)
